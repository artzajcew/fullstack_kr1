📚 Структура репозитория
Практическая работа №2
Серверное приложение на Express.js

CRUD операции для мебельного магазина

Маршруты: GET, POST, PATCH, DELETE для товаров

Практическая работа №3
Работа с JSON и внешними API

Тестирование API через Postman

Интеграция с открытыми API (погода, курсы валют)

Практическая работа №4
Клиент-серверное приложение (React + Express)

Интернет-магазин мебели с 10+ товарами

Функционал: просмотр, создание, редактирование, удаление товаров

Стек: React, SCSS, Axios, Express, nanoid, CORS

Структура клиента (pr4/client):
/src/components - переиспользуемые компоненты

/src/pages - страницы приложения

/src/api - настройка axios для запросов к серверу

Структура сервера (pr4/server):
app.js - основной файл сервера

API endpoints: /api/products

Практическая работа №5
Документирование API с помощью Swagger/OpenAPI

Интерактивная документация по адресу /api-docs

Автоматическая генерация спецификации из JSDoc-комментариев

🛠️ Технологии
Frontend: React, SCSS, Axios

Backend: Node.js, Express

База данных: in-memory (массив объектов)

Документация: Swagger (OpenAPI 3.0)

Генерация ID: nanoid

CORS: настройка для работы клиента и сервера

🚀 Запуск проекта
Сервер (backend)
bash
cd pr4/server
npm install
npm start
# Сервер запустится на http://localhost:3000
# Документация Swagger: http://localhost:3000/api-docs
Клиент (frontend)
bash
cd pr4/client
npm install
npm start
# Клиент запустится на http://localhost:3001
